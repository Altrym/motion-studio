# Creative Companion Skills

This folder bundles a curated subset of Hermes creative skills alongside `motion-studio`.

Included skills:

- `ascii-video/`
- `excalidraw/`
- `manim-video/`
- `p5js/`
- `popular-web-designs/`

Install each skill directory independently into your local skills home. For example:

```text
.claude/skills/ascii-video/
.claude/skills/excalidraw/
.claude/skills/manim-video/
.claude/skills/p5js/
.claude/skills/popular-web-designs/
```

The imported skills are vendored from Hermes Agent's `skills/creative` collection with small compatibility edits for local use. See `../THIRD_PARTY_NOTICES.md` for attribution and license details.
